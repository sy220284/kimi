## 波浪分析模块增强
升级了 Elliott Wave 算法，主要改进：

### 1. elliott_wave.py
- 新增 `VolumeAnalyzer` 成交量分析器
  - 各浪型量价特征识别 (浪1温和放量、浪3明显放量、浪5缩量背离)
  - 量价配合验证规则
- 新增 `FibonacciAnalyzer` 斐波那契分析器
  - 计算浪间比例关系 (浪2回撤、浪3vs浪1、浪4回撤)
  - 验证是否符合黄金分割规则
- 增强 `ElliottWaveAnalyzer`
  - 改进极值点识别算法，加入最小变动阈值过滤噪声
  - 推动浪和调整浪检测优化
  - 综合置信度计算 (原始置信度 + 量价 + 斐波那契)

### 2. wave_detector.py
- 增强版 `WaveDetector` 检测器
- 多维度信号生成: buy/sell/watch/strong_buy/strong_sell
- 风险收益比计算
- 详细的交易理由生成
- 量价和斐波那契对信号的增强/削弱机制

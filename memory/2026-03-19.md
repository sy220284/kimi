### 修复的问题（来自代码审查 B-06 ~ B-20）

#### 回测引擎（wave_backtester.py）
- **B-06** `avg_return_per_trade`：修复为 `mean(t.pnl_pct)` 各笔交易收益率均值
  - 原实现 `total_return_pct / 笔数` 无统计意义
- **B-17** Sharpe 无风险利率：补入 `rf = 3%/252` 日化扣减
  - 原实现假设无风险利率为零，Sharpe 虚高约 15-30%

#### 波浪分析层（unified_analyzer.py）
- **B-07** 置信度基础分：四处全部 `0.5 → 0.3`
  - 原来空结构信号也能自动通过 0.5 阈值，过滤形同虚设
- **B-08** 共振双重兜底：移除"全部过滤时回退原始列表"逻辑
  - 改为返回空列表，共振过滤真正生效
- **B-13** 信号静默丢弃：`else: continue → else: optimized_signals.append(sig)`
  - wave_structure=None 时信号不再无提示消失
- **B-15** C 浪量价索引偏移：`wave_a_idx = pivots[-4].idx`（真正 A 浪起点）
  - 原来用 `pivots[-3]` 偏晚一位，量价区间包含 B 浪数据
- **B-16** Fallback 索引越界：`len(df)-N → max(0, len(df)-N)`
- **B-18** 独立调用前瞻：`prices = detect_df['close'].values`
  - 与极值点索引对齐，消除独立调用时的轻微前瞻偏差

#### 入场质量优化器（entry_optimizer.py）
- **B-12** `final_score × 2.5`：删除错误放大系数
  - 原来乘以2.5后截断到1.0，等效门槛仅0.16，过滤形同虚设
  - 修复后 `min_quality_score` 从 0.40 提升至 0.55

#### 艾略特波浪（elliott_wave.py）
- **B-09** Flat 平台型调整浪：新增 `validate_flat()` + `_try_flat()`
  - 识别 Regular Flat / Expanded Flat / Running Flat 三种子型
  - A股最常见的 Expanded Flat（B浪突破A浪起点）优先识别
  - 集成到 `_detect_with_points()` 检测管线

#### 智能体层
- **B-20** RotationAgent 签名：去掉 `| None = None`，与 BaseAgent 契约一致
- **B-11** 轮动伪数据：完整重写 `rotation_analyst.py`
  - **主路径**：查询 `sw_industry_index`（申万行业指数），计算20/60日动量、相对强弱、趋势、波动率，输出行业轮动排名和配置建议
  - **回退路径**：行业表为空时按上市板块（科创/创业/沪主/深主）聚合全量个股动量（非原来仅5只样本）
